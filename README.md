# 🐝 ViUnAbejorro

Aplicación web interactiva con enfoque visual y experimental, desarrollada para explorar animaciones, interacción y experiencia de usuario en frontend.

---

## 🎯 Descripción

**ViUnAbejorro** es un proyecto orientado a la experimentación visual en el navegador, donde se busca generar una experiencia simple pero atractiva para el usuario.

El foco principal está en:

* Interacción en pantalla
* Feedback visual
* Fluidez en la UI

---

## 🚀 Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript (ES6+)

---

## ✨ Funcionalidades

* Interacción dinámica en pantalla
* Elementos visuales animados
* Experiencia responsive básica

---

## 🧠 Objetivo del proyecto

Este proyecto fue desarrollado para:

* Practicar manipulación del DOM
* Explorar animaciones en frontend
* Mejorar la experiencia de usuario
* Experimentar con ideas visuales simples

---

## ⚙️ Cómo ejecutar el proyecto

```bash
git clone https://github.com/VisF/ViUnAbejorro.git
cd ViUnAbejorro
```

Luego abrir el archivo `index.html` en el navegador.
